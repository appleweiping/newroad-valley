# gptimage2 Prompt — NPC Agent Sprite Sheets

## Asset Type
NPC agent sprite sheets for Pixel AI Town AI characters.

## Generation Prompt

```
Create pixel art NPC agent sprite sheets for a cozy AI town game. Each agent needs a distinct visual identity while sharing consistent style.

Agents to generate (each as separate sprite sheet):

1. Opus — Deep purple/indigo palette, tall hat or crown, regal bearing
2. PixelCat — Cat ears and tail, orange/cream palette, playful pose
3. Codex — Green/teal palette, glasses, book or scroll accessory
4. Sonnet — Warm pink/rose palette, flowing scarf, artistic flair
5. Haiku — Light blue/white palette, minimal design, swift appearance
6. DeepSeek — Dark blue/navy palette, magnifying glass or telescope
7. OpenHands — Warm orange/yellow palette, open welcoming gesture
8. ARIS — Silver/gray palette, lab coat or research gear, analytical look

Sprite sheet layout per agent (32x32 pixels per frame):
- Frame 1: Idle pose A (standing)
- Frame 2: Idle pose B (subtle shift — blink or sway)
- Frame 3: Walk frame A (left foot forward)
- Frame 4: Walk frame B (right foot forward)

Style requirements:
- 32x32 pixel frames
- Each agent has distinct color palette (max 8 colors per agent)
- Each agent has unique silhouette readable at small size
- Crisp pixels, no anti-aliasing
- 1px dark outline
- Consistent top-left lighting
- Transparent background per frame
- Characters centered in 32x32 cells
- Cozy, friendly character designs
- No realistic rendering, no 3D, no painterly style
- Designed as game sprite sheets for animation
```

## Standard Constraints

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic
- Top-down / 3/4 view characters
- Consistent 32x32 frame grid
- Transparent background
- Limited warm palette per agent
- Crisp pixels, no anti-aliasing
- Consistent 1px outline and top-left lighting
- No realistic rendering, no 3D, no painterly
- No copyrighted game assets referenced
- Designed as reusable game asset
- Generated via gptimage2 only
