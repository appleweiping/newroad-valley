# gptimage2 Prompt — Player Character Sprite Sheet

## Asset Type
Player-controlled character sprite sheet for Pixel AI Town.

## Generation Prompt

```
Create a pixel art character sprite sheet for the player character in a cozy AI town game. The character should be cute and anime-inspired.

Character design:
- Cute anime-inspired proportions (large head, small body)
- Long flowing hair (soft pastel color — lavender or light pink)
- Fantasy head accessory (small flower crown or star hairpin)
- Simple cozy outfit (hoodie or cardigan, comfortable)
- Warm, gentle, approachable feeling
- Tiny but readable silhouette at 32x32

Sprite sheet layout (32x32 pixels per frame):
Row 1: Idle facing down (2 frames — subtle breathing animation)
Row 2: Walk down (4 frames)
Row 3: Walk up (4 frames)
Row 4: Walk left (4 frames)
Row 5: Walk right (4 frames)

Style requirements:
- 32x32 pixel frames
- Limited warm palette (max 10 colors for character)
- Crisp pixels, no anti-aliasing
- 1px dark outline
- Consistent top-left lighting
- Transparent background per frame
- Character should be centered in each 32x32 cell
- Readable and charming at actual game size
- Soft, gentle feeling — not aggressive or edgy
- No realistic rendering, no 3D, no painterly style
- Designed as game sprite sheet for animation
```

## Standard Constraints

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic
- Top-down / 3/4 view character
- Consistent 32x32 frame grid
- Transparent background
- Limited warm palette
- Crisp pixels, no anti-aliasing
- Consistent 1px outline and top-left lighting
- No realistic rendering, no 3D, no painterly
- No copyrighted game assets referenced
