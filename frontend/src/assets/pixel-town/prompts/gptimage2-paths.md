# gptimage2 Prompt — Path & Road Tiles

## Asset Type
Path and road tileset for Pixel AI Town walkways and bridges.

## Generation Prompt

```
Create a pixel art path and road tileset for a cozy AI town game. The tileset should be arranged in a grid of 32x32 pixel tiles.

Include the following path types:
- Cobblestone path (straight, corners, T-junctions, crossroads)
- Dirt path (worn trail through grass, straight and curved)
- Wooden bridge (horizontal and vertical, with railings)
- Stone bridge (wider, decorative)
- Path edges and transitions to grass
- Stepping stones (scattered stone path)
- Brick walkway (town center style)

Style requirements:
- Top-down view
- 32x32 pixel grid per tile
- Limited warm palette (browns, grays, warm wood tones)
- Crisp pixels, no anti-aliasing
- 1px dark outline on path edges
- Consistent top-left lighting with subtle shadows
- Transparent background for overlay onto terrain
- Cozy village atmosphere
- Paths should feel well-worn and inviting
- No realistic rendering, no 3D effects, no painterly style
- Designed as seamlessly tiling game asset
```

## Standard Constraints

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic
- Top-down view
- Consistent 32x32 tile grid
- Transparent background where appropriate
- Limited warm palette
- Crisp pixels, no anti-aliasing
- Consistent 1px outline and top-left lighting
- No realistic rendering, no 3D, no painterly
- No copyrighted game assets referenced
- Designed as reusable, tileable game asset
- Generated via gptimage2 only
