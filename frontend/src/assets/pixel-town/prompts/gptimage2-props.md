# gptimage2 Prompt — Decoration Props

## Asset Type
Small reusable decoration props for Pixel AI Town map.

## Generation Prompt

```
Create pixel art decoration props for a cozy AI town game. Each prop should be a small, reusable sprite on transparent background.

Props to generate:
- Trees: Oak tree, pine tree, cherry blossom, small bush (32x32 to 32x64)
- Flowers: Flower patch, single flower, flower pot, sunflower (16x16 to 32x32)
- Fences: Wooden fence segment, gate, stone wall segment (32x32)
- Lamps: Street lamp, hanging lantern, ground torch (16x32 to 32x32)
- Benches: Wooden bench, stone bench (32x16 to 32x32)
- Signs: Wooden signpost, directional sign, notice board (32x32)
- Fountains: Small fountain, birdbath (32x32 to 48x48)
- Misc: Barrel, crate, well, mailbox, trash can, rock pile (16x16 to 32x32)

Style requirements:
- Top-down / 3/4 view (matching building perspective)
- 32x32 base grid alignment
- Limited warm palette (greens, browns, warm grays)
- Crisp pixels, no anti-aliasing
- 1px dark outline
- Consistent top-left lighting
- Transparent background
- Each prop should be readable and recognizable at game zoom
- Cozy village atmosphere with lived-in details
- No realistic rendering, no 3D effects, no painterly style
- Designed as reusable, placeable game props
```

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic

- Consistent with 32x32 tile grid
- Transparent background
- Limited warm palette
- Crisp pixels, no anti-aliasing

- No copyrighted game assets referenced
