# gptimage2 Prompt — Terrain Tileset

## Asset Type
Terrain tileset for Pixel AI Town ground tiles.

## Generation Prompt

```
Create a pixel art terrain tileset for a cozy AI town game. The tileset should be arranged in a grid of 32x32 pixel tiles.

Include the following terrain types:
- Grass (light green, medium green, dark green variants)
- Dirt (dry earth, tilled soil)
- Sand (beach sand, desert sand)
- Water (still water, flowing water, shoreline edges)
- Grass-to-dirt transitions (all 4 edges + corners)
- Grass-to-water transitions (all 4 edges + corners)
- Sand-to-water transitions (all 4 edges + corners)

Style requirements:
- Top-down view
- 32x32 pixel grid per tile
- Limited warm palette (earth tones, greens, blues)
- Crisp pixels, no anti-aliasing
- 1px dark outline on transition edges only
- Consistent top-left lighting
- Transparent background for overlay tiles
- Cozy, inviting atmosphere
- No realistic rendering, no 3D effects, no painterly style
- Designed as seamlessly tiling game asset
```

## Standard Constraints

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic
- Top-down view
- Consistent 32x32 tile grid
- Transparent background where appropriate
- Limited warm palette
- Crisp pixels, no anti-aliasing
- Consistent 1px outline and top-left lighting
- No realistic rendering, no 3D, no painterly
- No copyrighted game assets referenced
- Designed as reusable, tileable game asset
- Generated via gptimage2 only
