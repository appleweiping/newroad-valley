# gptimage2 Prompt — Building Sprites

## Asset Type
Building sprites for Pixel AI Town structures.

## Generation Prompt

```
Create pixel art building sprites for a cozy AI town game. Each building should be a self-contained sprite.

Buildings to generate (one per image, or arranged on a sheet):
- Town Hall: Grand central building, clock tower, warm stone and wood, 128x128px
- Memory Library: Cozy library with glowing windows, bookshelves visible, 128x128px
- Skill Workshop: Rustic workshop with tools, anvil, workbench visible, 96x96px
- Dream Garden: Greenhouse/garden pavilion with flowers and vines, 128x128px
- DevTools Lab: Modern-ish lab with screens and gadgets, 96x96px
- Resource Market: Open market stall with awning and goods, 96x96px
- Knowledge Tower: Tall narrow tower with observatory dome, 128x128px
- Agent Homes: Cozy row of small houses with unique doors, 96x96px
- Plaza: Open stone plaza with fountain centerpiece, 128x128px

Style requirements:
- 3/4 top-down view (slightly elevated perspective)
- Warm wood, stone, and brick materials
- Glowing windows (warm yellow/orange light)
- Limited warm palette per building (max 12 colors)
- Crisp pixels, no anti-aliasing
- 1px dark outline
- Consistent top-left lighting with cast shadows
- Transparent background
- Each building should be distinct and recognizable at small size
- Cozy fantasy village aesthetic
- No realistic rendering, no 3D, no painterly style
- Designed as standalone game sprite
```

## Standard Constraints

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic
- 3/4 top-down view
- Consistent scale relative to 32x32 tile grid
- Transparent background
- Limited warm palette
- Crisp pixels, no anti-aliasing
- Consistent 1px outline and top-left lighting
- No realistic rendering, no 3D, no painterly
- No copyrighted game assets referenced
- Designed as reusable game asset
- Generated via gptimage2 only
