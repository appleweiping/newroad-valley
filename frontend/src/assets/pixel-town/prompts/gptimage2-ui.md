# gptimage2 Prompt — Pixel UI Elements

## Asset Type
UI elements for Pixel AI Town game interface.

## Generation Prompt

```
Create pixel art UI elements for a cozy AI town game interface. All elements should be nine-slice compatible where applicable.

UI elements to generate:

Panels:
- Dialog box (dark semi-transparent background, warm border, 3px border)
- Tooltip panel (smaller, pointed, warm accent border)
- Inventory panel (grid-ready interior, ornate corners)
- Status panel (compact, rounded corners)

Buttons:
- Primary button (warm wood/gold, pressed and unpressed states)
- Secondary button (stone/gray, pressed and unpressed states)
- Close button (small X, circular, red accent)
- Arrow buttons (left, right, up, down)

Nameplates:
- NPC nameplate (dark bg, colored border matching agent palette)
- Player nameplate (slightly different style to distinguish)

Log/Chat:
- Chat bubble (speech, left-pointing tail)
- Thought bubble (cloud shape, dots trail)
- System message box (subtle, less prominent)

Status Icons (16x16):
- Heart (health/mood)
- Star (skill/level)
- Book (knowledge)
- Lightning (energy)
- Clock (time)
- Gear (settings)

Style requirements:
- Dark theme base (deep navy/charcoal backgrounds)
- Warm accent colors (gold, amber, soft orange borders)
- Nine-slice compatible panels (clear corners, edges, center)
- Crisp pixels, no anti-aliasing
- 1px borders, consistent corner radius
- Subtle inner glow or highlight on interactive elements
- Transparent background for overlay elements
- Readable at various UI scales
- Cozy fantasy RPG UI aesthetic
- No realistic rendering, no 3D, no painterly style
- Designed as reusable UI components
```

## Standard Constraints

- Original pixel art asset — no copyrighted game assets
- Cozy AI town aesthetic
- Consistent pixel grid
- Transparent background where appropriate
- Dark theme with warm accents
- Crisp pixels, no anti-aliasing
- Consistent 1px outline and lighting
- No realistic rendering, no 3D, no painterly
- No copyrighted game assets referenced
- Designed as reusable, nine-slice compatible game UI
- Generated via gptimage2 only
